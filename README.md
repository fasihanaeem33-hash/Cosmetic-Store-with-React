 # Cosmetic Store

 A modern React store demo built with Vite and Tailwind CSS. This project features product browsing, category filtering, cart navigation, and responsive pages for Home, Products, Categories, About, Contact, and Cart.

 ## Features

 - React 19 + Vite application
 - Client-side routing with `react-router-dom`
 - Product listing and category pages
 - Shared layout with `Navbar` and `Footer`
 - Tailwind CSS styling via `@tailwindcss/vite`

 ## Project Structure

 - `src/App.jsx` — application router and layout
 - `src/components/` — shared UI components
 - `src/pages/` — route pages for Home, Products, Categories, About, Contact, Cart
 - `src/data/products.js` — sample product data

 ## Scripts

 - `npm run dev` — start the local development server
 - `npm run build` — build the production site
 - `npm run preview` — preview the production build locally
 - `npm run lint` — run ESLint across the project

 ## Installation

 1. Install dependencies:
    ```bash
    npm install
    ```
 2. Start the dev server:
    ```bash
    npm run dev
    ```

 ## Dependencies

 - `react`
 - `react-dom`
 - `react-router-dom`
 - `tailwindcss`
 - `@tailwindcss/vite`

 ## Notes

 - The sample product catalog is defined in `src/data/products.js`.
 - Routes are configured in `src/App.jsx`.
 - Update or extend pages and product data as needed for your store.
